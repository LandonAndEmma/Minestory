# Minestory
A Minecraft inspired choose your own adventure game.

Requirments:\n
Python 3.9\n
Random (For Random Generation) [Automatic]\n
Tkinter (For Windows) [Automatic]\n
Pygame (For Music) [Manual PIP Install]\n
